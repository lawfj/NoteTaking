# NoteTaking
记笔记1.0版本
记笔记是一款简美且好用的笔记应用，回归笔记的文字时代，方便你随时随地记录点滴 。 
【主要功能】 
1. 用文字记录身边随时发生的事情。 
2. 快速对笔记进行搜索。 
3. 可设置侧滑栏位置，操作更方便。 
4. 两种笔记展示样式，更美观。 
【界面设计】 
遵循Google Material Design，悬浮滑动按钮，上滑隐藏导航条。

####记笔记1.3版本
添加更换主题功能，提供十二款混搭纯色配色，一定有你的菜！
优化备份功能为用户需要时点击备份
优化了代码逻辑
